- Python 3.10.6

- Streamlit

### Commands Used
- source my_env/bin/activate (to activate python env)
- pip install streamlit Pillow
- streamlit run app.py
- pip install pipreqs
- pipreqs --encoding=utf8 (generate requirements.txt)
